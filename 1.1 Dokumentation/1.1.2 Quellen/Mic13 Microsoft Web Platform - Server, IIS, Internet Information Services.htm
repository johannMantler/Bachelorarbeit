Die in Windows Server 2012 R2 integrierten Internetinformationsdienste (IIS) 8.5 bieten eine sichere und leicht zu verwaltende Plattform, die modular aufgebaut und jederzeit erweiterbar ist. Stellen Sie damit Ihre Websites, -services und -anwendungen zuverlässig im Netz bereit. Mit IIS haben Sie viele Wahl- und Steuerungsmöglichkeiten, ohne dass Sie bei der Zuverlässigkeit und Sicherheit Einbußen hinnehmen müssen. Verändern Sie vorhandene Eigenschaften oder fügen Sie neue Funktionen wie etwa Routing von Anwendungsanforderungen und Lastenausgleich mit den kostenfreien IIS-Erweiterungen hinzu. Erhöhen Sie die Websicherheit, indem Sie die von einem Server hinterlassen Daten in einem Netzwerk auf ein Minimum reduzieren und zusätzlich Anwendungen automatisch isoliert ausführen. Verwenden Sie ASP.NET- und PHP-Webanwendungen gleichzeitig auf dem gleichen Server. Das alles und noch viel mehr bieten Ihnen die IIS. Erfahren Sie mehr über die ersten Schritte auf der IIS.NET-Website.

Lernen Sie IIS 8.5 schätzen

Sofort einsetzen. Kostenfrei nutzen.

Alles, was Sie für die Ausführung und Bereitstellung Ihrer Webanwendungen mit IIS benötigen, erhalten Sie unkompliziert mit dem Microsoft-Webplattform-Installer. Die einfach zu bedienende, optimierte Installationsanwendung sorgt dafür, dass Ihr Entwicklungs- oder Hostingcomputer sofort einsatzbereit ist. Sie enthält die aktuelle Version der IIS und deren Erweiterungen. Zudem erhalten Sie so auch die aktuellen Versionen der Microsoft Web-Plattform mit allen Frameworks, Datenbanken und Entwicklungstools. Das Gute daran: Sie erhalten alles kostenfrei! Laden Sie Ihre Version jetzt herunter.

Alles wird Eins.

Von ASP.NET bis hin zu PHP bieten IIS einen leistungsstarken und flexiblen Webserver für die beliebtesten Webanwendungen, wie etwa Umbraco, WordPress und Drupal. Dank vollständiger Integration in die Pipeline für die Anforderungsverarbeitung, das Konfigurationssystem und die Verwaltungskonsole ist ASP.NET-Hosting noch einfacher und leistungsfähiger. Erfahren Sie mehr über PHP unter Windows.

Erhalten Sie mehr für Ihr Geld.

IIS 8.5 bieten verschiedene neue Features, die speziell auf umfangreiche Hosting-Umgebungen abzielen. Durch die dynamische Aktivierung von Websites und das Aussetzen im Leerlauf befindlicher Arbeitsprozesse lässt sich die Ressourcenauslastung reduzieren. So können Sie die Dichte der Websites erhöhen und die Gesamtbetriebskosten reduzieren. Zudem ermöglichen die neuen, erweiterten Protokollierungsoptionen eine verbesserte Datenerfassung und Problembehandlung. Erfahren Sie mehr über die neuen Features in IIS 8.5.

Verwenden Sie nur das, was Sie benötigen.

Die IIS sind komplett modular auf den vielfältig erweiterbaren APIs aufgebaut. Dank dieser Architektur sind Sie in der Lage, sehr einfach IIS-Komponenten hinzuzufügen, zu entfernen oder zu ersetzen und so mit angepassten Modulen nahezu jeden Kundenwunsch zu erfüllen. Verwenden Sie nur das, was Sie benötigen. Verändern Sie vorhandene Eigenschaften oder fügen Sie neue Funktionen wie etwa Routing von Anwendungsanforderungen und URL-Neuschreibung mit den kostenfreien IIS-Erweiterungen hinzu.

Leistung

Beschleunigen Sie Ihre Website mit dem integrierten dynamischen Cache-Verfahren und verbesserter Datenkompression. IIS bieten durch eine leistungsfähige HTTP-Kompression sowie eine tiefere Integration in die Abfrageprozesse vom Windows-Kernel an SSL-geschützte Websites eine sehr hohe Leistung, wenn es um die Verarbeitung statischer und dynamischer Webinhalte geht. Implementieren Sie eine skalierbare Web-Infrastruktur mit einem HTTP-basierten Lastenausgleich und einer intelligenten Handhabung von Anforderungen und Weiterleitungen.

Sicherheit

Mit IIS erhöhen Sie automatisch die Sicherheit, weil Ihre Webserver damit nur so viele Spuren wie nötig in einem Netzwerk hinterlassen und Anwendungen automatisch gekapselt ausführen. IIS bieten leistungsfähiges Sandboxing durch CPU-Drosselung, wodurch die Wahrscheinlichkeit reduziert wird, dass Anwendungen ausbrechen. Außerdem isolieren IIS Anwendungen besser, da sie ihnen bei der Ausführung für alle Arbeitsprozesse eine einzigartige Identifikationsnummer und eine eigene Sandbox-Konfiguration zuweisen. So minimieren Sie Sicherheitsrisiken.
